But here's a sample:

The press website RenewEconomy is pretty good at keeping up-to-date maps of renewable projects, in operation or in construction; arguably better than some of the official database.

With Python and the Selenium library, we can scrape this public data and build our own database, complete with the name, coordinates, power, technology, owner, etc., of each power plant.

Then we can import this database to use in our GIS tool, such as ArcMap.
